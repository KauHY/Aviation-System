权限设置指南

如果测试失败，请按照以下步骤检查权限设置：

浏览器权限设置：

  • Chrome/Edge：点击地址栏左侧的锁图标 → 选择"站点设置" → 将摄像头和麦克风设置为"允许"
  • Firefox：点击地址栏左侧的信息图标 → 选择"更多信息" → 点击"权限"标签 → 允许使用摄像头和麦克风

系统权限设置：

  • Windows：开始菜单 → 设置 → 隐私 → 摄像头/麦克风 → 确保"允许应用访问您的摄像头/麦克风"已开启
  • macOS：苹果菜单 → 系统偏好设置 → 安全性与隐私 → 隐私 → 摄像头/麦克风 → 确保浏览器已被勾选

常见问题：

  • 确保没有其他应用正在使用摄像头或麦克风
  • 重启浏览器后重试
  • 尝试使用其他浏览器
  • 检查设备驱动是否正常
  • 确保网络连接稳定
